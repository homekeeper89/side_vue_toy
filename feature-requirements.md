# 요구사항 리스트

## 사용자(User) 관련

### 1. 사용자 등록

1. p 필요 정보를 통해 회원 가입이 가능하다
   > 이메일, 비밀번호, 닉네임
   > 1.1 회원 가입 기능은 차후에 git 연동이 가능 하게 변경 필요
2. 이메일 인증 기능?
3. 이메일 중복 검사가 가능해야한다.
   2.1 이메일 중복은 자동으로 이루어져야함
   2.2 이메일 중복에 해당되면 "이미 등록되어 있는 이메일 입니다(빨간색)" 안내
   > 이메일 찾기 기능 안내
4. 비밀번호는 재입력을 통해 정확하게 입력 하도록 유도
   p > 비밀번호가 다를 경우 "서로 다른 비밀번호 입니다" 안내
5. 개인정보 이용 동의 체크 박스
